Ben Kimim ?

2023 yılında Erciyes Üniversitesi Bilgisayar Mühendisliği bölümünü kazandım. Bir yıl İngilizce hazırlık eğitimimi tamamladıktan sonra bölüm derslerime başladım. 2025 yılı itibariyle 1. sınıf eğitimimi başarıyla bitirdim. Bu süreçte yalnızca akademik derslerle sınırlı kalmayarak, Udemy üzerinden “Sıfırdan İleri Düzey Web Geliştirme” kursunu tamamladım ve öğrendiklerimi pekiştirmek için kendi web sitemi tasarlamaya başladım. Hedefim; öğrendiklerimi gerçek projelere aktararak sürekli gelişmek ve profesyonel yazılım dünyasında sağlam adımlarla ilerlemek!

Bildiklerim

Temel programlama ve algoritma mantığını iyi kavradım. C ve C# .NET Framework ile uygulama geliştirme deneyimim var. Ayrıca web teknolojileri olarak HTML, CSS, JavaScript ve TypeScript biliyorum. Bu bilgi birikimiyle hem backend hem frontend projelerde çalışmaya hazırım.
